# Git Cheat Sheet 🚀

| Action | Commande |
|---|---|
| Initialiser Git | `git init` |
| Voir l’état du dépôt | `git status` |
| Ajouter un fichier | `git add fichier.txt` |
| Ajouter tous les fichiers | `git add .` |
| Faire un commit | `git commit -m "message"` |
| Voir l’historique | `git log` |
| Historique compact | `git log --oneline` |
| Voir les branches | `git branch` |
| Créer une branche | `git branch nom-branche` |
| Changer de branche | `git switch nom-branche` |
| Créer + changer de branche | `git switch -c nom-branche` |
| Fusionner une branche | `git merge nom-branche` |
| Supprimer une branche | `git branch -d nom-branche` |
| Ajouter un dépôt distant | `git remote add origin URL` |
| Envoyer sur GitHub | `git push -u origin main` |
| Récupérer les modifications | `git pull` |
| Cloner un dépôt | `git clone URL` |
| Retirer du staging | `git reset fichier.txt` |
| Reset soft | `git reset --soft ID_COMMIT` |
| Reset hard | `git reset --hard ID_COMMIT` |
| Modifier le dernier commit | `git commit --amend --no-edit` |
| Voir toutes les actions Git | `git reflog` |
| Voir l’auteur des lignes | `git blame fichier.txt` |
